## Features

- **Interactive Visualizations**: Dynamic charts and graphs that respond to user filters
- **Data Modeling**: Efficient data relationships and calculated measures using DAX
- **Clean Design**: Focus on clarity and ease of interpretation
- **Cross-filtering**: Click on any visual to filter related data across the dashboard
