## Overview

An interactive Power BI dashboard showcasing data visualization best practices and business intelligence capabilities. The dashboard provides actionable insights through dynamic visualizations and intuitive navigation.

## Features

- **Interactive Filtering** — Dynamic charts that respond to user selections
- **Data Modeling** — Efficient relationships and calculated measures using DAX
- **Clean Design** — Focus on clarity and ease of interpretation
- **Cross-filtering** — Click any visual to filter related data across the dashboard

## Technical Skills

**DAX** — Custom calculated columns, measures, time intelligence functions, and conditional formatting.

**Data Modeling** — Star schema implementation, relationship management, and performance optimization.

**Visualization** — Consistent color schemes, appropriate chart selection, and clear labeling.
